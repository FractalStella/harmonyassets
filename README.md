
# Harmony Assets
Aqui se podran descargar los assets necesarios para OBS




## Descargas
Clickea en el boton verde que dice ```Code``` y luego clickea en ```Download Zip```.
Ese Zip lo puedes extraer en tu folder local de Harmony.
